Slide 08 — CTE Query
The final score: combining all 5 signals into one number
A CTE combines consistency (40%), YoY improvement (30%), and value per match (30%) into a single final_score. The SQL did the scouting.
WITH batsman_scores AS (
  SELECT Player, Team,
  ROUND(...) AS consistency_score,
  ROUND(b19.Avg - b18.Avg) AS improvement,
  ROUND(...) AS value_index
  FROM batsmen_2019 b19
  JOIN batsmen_2018 b18 ON b19.Player = b18.Player
)
SELECT *, ROUND(
  (consistency_score × 0.4)
  + (improvement × 0.3)
  + (value_index × 0.3), 2) AS final_score
FROM batsman_scores
ORDER BY final_score DESC;
